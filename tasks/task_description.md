- Reads the video file
- Detects motion (basic or ML)
- Assigns each frame a sperm count by motility class

|---------|------|------|----------|
| 1       |  5   |  3   |    2     |
| 2       |  6   |  1   |    3     |
| ...     | ...  | ...  |   ...    |

(Bonus) Draw bounding boxes and save a new video if you can.
